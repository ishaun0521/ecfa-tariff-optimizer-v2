網站改版歷程

目前版本 v0.5.0 更新時間：2026-03-12（台灣時間）

這一頁用來記錄本平台每次改版增加了哪些功能、改善哪些內容，讓使用者與合作夥伴可以直接看到平台的演進歷程。

改版時間線

v0.5.0｜改版歷程頁 + 版本資訊2026-03-12

  • 新增 改版歷程頁，可直接查看每次版本更新內容。
  • 首頁、官方法規來源頁、稅則教學頁都加入 版號與更新時間。
  • 新增「改版歷程」入口，方便使用者追蹤平台演進。

v0.4.4｜範例 BOM 下載 + 食品類提示2026-03-12

  • 在 BOM 上傳區新增 3 個 範例商品 CSV 下載。
  • 讓使用者可以先下載範例格式，再改成自己的 BOM 後上傳。
  • 首頁新增提示：目前平台主要聚焦 食品類，其他類別將陸續增加。

v0.4.3｜入站免責聲明確認彈窗2026-03-12

  • 使用者一進首頁即顯示 邊界說明 / 免責聲明確認彈窗。
  • 必須勾選並按下「我已了解」後，才能開始操作平台。
  • 彈窗內可直接前往官方法規來源頁與稅則教學頁。

v0.4.2｜品牌導流與所有權標示2026-03-12

  • 平台新增 所有權標示：歸屬於 caio.com.tw 陳以軒。
  • 首頁、法規來源頁、教學頁加入前往 caio.com.tw 的導流入口。
  • 所有權區塊移至頁面最底部，讓使用者先看完內容再看到導引。

v0.4.1｜法規來源頁 + 稅則教學頁2026-03-12

  • 新增 /legal-sources：展示台灣與中國端官方法規來源及 URL。
  • 新增 /tariff-guide：教學頁說明稅則判定流程、ECFA 清單、原產地與文件四層關係。
  • 加入流程圖（SVG），方便用於對外說明與展示。

v0.4.0｜稅則前置判定模組2026-03-12

  • 新增 /classify API，支援稅則前置判定（pre-classification）。
  • 將稅則候選號列、缺資料、風險提示、人工覆核條件接入 `/analyze` 與 `/optimize`。
  • 加入 `customs_boundary_notice`，明確聲明本平台非最終海關裁定。

v0.3.0｜案例級商業邏輯 MVP2026-03-12

  • 三個食品案例模板上線：珍珠奶茶、鳳梨酥、芋頭酥。
  • 新增 `case_insights`、`key_risk_materials`、`recommended_next_checks`。
  • 候選方案與案例情境連動，不再只是 generic heuristic。

v0.2.0｜前端 MVP + 線上展示2026-03-12

  • 首頁改成可直接操作的 Web UI，不再只是 API JSON。
  • 分析與最佳化結果頁改為可讀性較高的產品化卡片式展示。
  • Zeabur 線上站可打開首頁、健康檢查與 API docs。

v0.1.0｜專案初始化2026-03-11

  • 建立 FastAPI 後端 MVP。
  • 提供 `/analyze`、`/optimize`、`/upload-bom` 等基礎 API。
  • 建立食品類 BOM 測試資料與 ECFA 官方研究來源清單。

平台所有權與延伸服務
